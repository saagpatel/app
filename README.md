## Prerequisites

- macOS with Xcode 16 or later installed
- An Apple Developer account (free tier sufficient for Simulator builds)
- iOS 26.2 SDK (ships with Xcode 16)

---

## Getting Started

1. **Clone the repository**

   ```bash
   git clone https://github.com/saagpatel/app.git
   cd app
   ```

2. **Open in Xcode**

   ```bash
   open app.xcodeproj
   ```

3. **Select a scheme and destination**

   In the Xcode toolbar, choose the `app` scheme and pick a Simulator or connected device running iOS 26.2+.

4. **Build and run**

   Press `Cmd + R` or choose **Product → Run**.

No package dependencies or additional configuration steps are required.
