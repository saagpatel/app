> Native SwiftUI app for iPhone, iPad, and Apple Vision Pro.

A native iOS application built with SwiftUI targeting iOS 26.2+, with support for iPhone, iPad, and Apple Vision Pro.

## Features

- **Cross-device** — Runs on iPhone, iPad, and Apple Vision Pro from a single codebase
- **SwiftUI** — Declarative UI with full platform idioms
- **iOS 26.2** — Built against the latest SDK

## Quick Start

```bash
git clone https://github.com/saagpatel/app.git
cd app
open app.xcodeproj
```

Select the `app` scheme in Xcode, choose a Simulator or connected device running iOS 26.2+, and press Run.
